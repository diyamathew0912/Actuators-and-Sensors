# NeoPixel Strip (4 LEDs) using Arduino

## Aim

To control a NeoPixel Strip with 4 LEDs using Arduino Uno and display different colors.

## Components Used

* Arduino Uno
* NeoPixel Strip (4 LEDs)
* Jumper Wires

## Connections

* IN → D6
* Power (+) → 5V
* Ground (-) → GND
* OUT → Not Connected

## Working

The Arduino sends data to the NeoPixel strip through the IN pin. The LEDs glow in different colors such as red, green, and blue. After displaying the colors, the LEDs turn off and the sequence repeats continuously.

## Arduino Code

Uses the Adafruit NeoPixel library to control the LEDs individually and create color-changing effects.

## Applications

* Decorative Lighting
* Smart Home Lighting
* LED Animations
* Wearable Electronics
* Display Systems

## Result

The NeoPixel strip successfully displayed multiple colors and lighting effects under Arduino control.
