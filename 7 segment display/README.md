# 7-Segment Display Counter using Arduino

## Aim

To display numbers from 0 to 9 sequentially on a 7-segment display using Arduino.

## Components Used

* Arduino Uno
* Common Cathode 7-Segment Display
* 7 × 220Ω Resistors
* Breadboard
* Jumper Wires

## Connections

* Segment a → D2
* Segment b → D3
* Segment c → D4
* Segment d → D5
* Segment e → D6
* Segment f → D7
* Segment g → D8
* Common Cathode → GND

## Working

The Arduino controls the segments of the 7-segment display to show digits from 0 to 9. Each number is displayed for one second before moving to the next number. After displaying 9, the sequence repeats.

## Applications

* Digital Clocks
* Electronic Counters
* Scoreboards
* Calculator Displays
* Embedded Systems

## Result

The 7-segment display successfully displays numbers from 0 to 9 repeatedly.
