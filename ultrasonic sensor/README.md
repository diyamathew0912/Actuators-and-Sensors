# Ultrasonic Distance Sensor (HC-SR04) with Arduino

## About
This project measures the distance of objects using an ultrasonic sensor and Arduino.

## Components Used
- Arduino Uno  
- HC-SR04 Ultrasonic Sensor  
- Jumper wires  

## Connections
- VCC → 5V  
- GND → GND  
- TRIG → Digital Pin 9  
- ECHO → Digital Pin 10  

## Working
The sensor sends sound waves and calculates distance based on the time taken for the echo to return.

## Output
Distance is displayed in centimeters on the Serial Monitor.

## Applications
- Obstacle detection  
- Smart parking systems  
- Robotics projects
