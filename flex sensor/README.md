# Flex Sensor with Arduino

## About
This project measures bending using a flex sensor and displays values in Serial Monitor.

## Components
- Arduino Uno  
- Flex Sensor  
- 10k Resistor  

## Connections
- Flex sensor → A0 and 5V  
- 10k resistor → A0 to GND  

## Working
When the sensor bends, resistance changes and Arduino reads different values.

## Output
- Straight → Low/medium value  
- Bent → Higher value  

## Applications
- Gesture control  
- Robotics  
- Wearable devices
