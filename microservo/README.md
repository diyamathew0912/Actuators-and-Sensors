# Micro Servo Motor Project (Arduino)

## Description
This project demonstrates controlling a micro servo motor using Arduino UNO. The servo rotates between 0° and 180° based on programmed instructions.

## Components Used
- Arduino UNO
- SG90 Micro Servo Motor
- Jumper wires

## Working Principle
Arduino sends PWM signal through a digital pin. The servo interprets the signal and moves to the required angle.

## Wiring
- Red → 5V
- Black → GND
- Yellow → Pin 9

## Output
Servo rotates between set angles or sweeps smoothly depending on code.
