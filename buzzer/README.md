# Piezo Buzzer using Arduino

## Components Used
- Arduino Uno
- Piezo Buzzer
- Jumper Wires

## Connections
- Buzzer (+) → D8
- Buzzer (-) → GND

## Working
The Arduino turns the buzzer ON for one second and OFF for one second repeatedly.

## Applications
- Alarm Systems
- Door Bells
- Warning Indicators
- Notification Systems
