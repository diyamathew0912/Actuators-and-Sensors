# Push Button Doorbell

## Description
This project uses a push button to control a buzzer. When the button is pressed, the buzzer produces a sound.

## Components Required
- Arduino Uno
- Push Button
- Piezo Buzzer
- Breadboard
- Jumper Wires

## Connections

### Push Button
- One pin → D2
- Other pin → GND

### Buzzer
- Positive (+) → D8
- Negative (-) → GND

## Working Principle
When the button is pressed, Arduino activates the buzzer. When released, the buzzer stops.

## Applications
- Doorbells
- Alert systems
- Simple alarms
