# Tilt Switch (2-Pin) with Arduino

## About
This project detects tilt using a simple 2-pin tilt switch and activates a buzzer.

## Components
- Arduino Uno  
- 2-Pin Tilt Switch  
- Buzzer  
- Jumper wires  

## Connections
- Tilt switch → D2 & GND  
- Buzzer → D9 & GND  

## Working
When the sensor tilts, the switch closes/open circuit changes and Arduino detects it.

## Output
- Tilt detected → Buzzer ON  
- Normal → Buzzer OFF
