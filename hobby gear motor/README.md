# Hobby Gear Motor using Arduino

## Aim

To control a hobby gear motor using Arduino Uno.

## Components Used

* Arduino Uno
* Hobby Gear Motor
* Jumper Wires

## Connections

* Motor Positive (+) → 5V
* Motor Negative (-) → GND

## Working

The hobby gear motor receives power from the Arduino and rotates continuously when the simulation starts.

## Applications

* Robotic Cars
* Conveyor Systems
* Automatic Machines
* Educational Robotics Projects

## Result

The hobby gear motor rotated successfully when connected to the power supply.
