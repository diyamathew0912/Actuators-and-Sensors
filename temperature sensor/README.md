# TMP36 Temperature Sensor

## Purpose
Measures temperature in degrees Celsius.

## Components Required
- Arduino Uno
- TMP36 Temperature Sensor

## Connections
Pin 1 -> 5V
Pin 2 -> A0
Pin 3 -> GND

## Working
The TMP36 outputs a voltage proportional to temperature.
Arduino reads the voltage and converts it into Celsius.

## Output
Temperature values are displayed in the Serial Monitor.
