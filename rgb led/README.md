# RGB LED Color Control using Arduino

## Aim
To control an RGB LED using Arduino.

## Components
- Arduino Uno
- RGB LED
- 3 × 220Ω Resistors
- Jumper Wires

## Connections
- Red Pin → D11
- Green Pin → D10
- Blue Pin → D9
- Common Pin → GND

## Working
The RGB LED changes color every second by turning ON the Red, Green, and Blue LEDs one after another.

## Applications
- Decorative Lighting
- Status Indicators
- Smart Lighting Systems
- Arduino Learning Projects
